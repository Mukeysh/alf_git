This widget allows you to display any piece of content from your site as a box. Start typing in the title of the content you want to display in the 'Select a node' box, and OpenScholar will find it for you.

Selecting Teaser for the Display Mode will only show a small portion of the post. This is suitable for most situations you would use this in. 'Full Node' will display the entire post as if it were the only thing on the page.

This is widget is best used on the front page, in one of the content regions.
